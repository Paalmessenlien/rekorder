# NBF rekord-typer

Kart over alle rekord-typer som kan registreres på
[rekord.bueskyting.no](https://rekord.bueskyting.no/), koblet mot
*Avstandstabell-mesterskap-skive* (NBF, 2024-11).

## Innhold

| Fil | Beskrivelse |
|---|---|
| [`index.html`](index.html) | Statisk web-app: oversikt, divisjoner, rundetyper, klasser, avstandstabell, fullstendig liste over rekord-typer (2000 stk.), og slå-opp-verktøy. |
| [`rekord-typer.md`](rekord-typer.md) | Teknisk dokumentasjon (norsk) — kilder, dimensjonsmodell, mapping mellom site og PDF, spesialtilfeller, verifikasjon. |
| [`rekord-typer.csv`](rekord-typer.csv) | Komplett enumerasjon: 2000 rader, én per (divisjon × klasse × rundetype × aspekt). |
| [`Avstandstabell-mesterskap-skive-2024-11.pdf`](Avstandstabell-mesterskap-skive-2024-11.pdf) | Kildedokument fra NBF (november 2024). |
